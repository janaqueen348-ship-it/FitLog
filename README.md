# FitLog 🏋️‍♀️

FitLog هو تطبيق ويب لمتابعة اللياقة البدنية، مصمم لمساعدة المستخدمين على تتبع أهدافهم الرياضية بسهولة.

## المميزات:
- تصميم تفاعلي وجذاب.
- تتبع التمارين اليومية.
- مبني باستخدام أحدث تقنيات الويب.

## التقنيات المستخدمة:
- **HTML5** لهيكلة الصفحات.
- **CSS3** لتصميم واجهة مستخدم مريحة وجذابة.
- **JavaScript** لإضافة الوظائف التفاعلية للتطبيق.
